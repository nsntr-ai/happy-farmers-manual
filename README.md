# Happy Farmers Manual

Repositori ini menyimpan dokumen panduan resmi untuk aplikasi Happy Farmers.

## Struktur Direktori

- `prompts/`: Berisi prompt yang digunakan untuk memandu AI dalam membantu merencanakan dan menulis panduan penggunaan sistem.
- `plan/`: Berisi cetak biru kerangka kerja (Outline) untuk seluruh modul di dalam sistem.
- `modules/`: Berisi kumpulan Markdown manual/buku panduan terperinci, dibagi per modul sistem untuk kemudahan pemeliharaan dan navigasi.
